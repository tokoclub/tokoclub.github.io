【精品购物指南】INTO1 追梦旅行，向下一个目标出发！

· 2022-04-25 · # 杂志 # INTO1 # 采访

成长，是风浪与苦行并存的涅槃蜕变，也是向着未来的冲天之志。
他们迫不及待地向着下一个目标出发，属于INTO1的追梦旅行，依旧未完待续….…

去年春天，一群来自五湖四海的少年们自海上启程，四时更替，朝暮轮转，转眼时间又进一年，他们相伴走过了又一个春夏秋冬，体会了无数的喜怒哀乐，也见证了彼此的成长。在国际男子唱演组合INTO1成团一周年之际，《精品购物指南》和少年们一起用影像记录下了这难忘的一刻，为他们即将到来的生日送上祝福。

INTO1为期两年的探索旅程分为“海、陆、空、宇宙”四个内容版块，首张音乐EP《风暴眼》讲述的是为海洋而集结的冒险故事；去年年底，他们推出了第二张EP《万里》，去亲历中国古老而又厚重的土地，去佛山体验岭南舞狮文化，去宁夏感受古丝绸之路的一环，前往重庆，去展现当代中国的一些烟火气。

接下来，少年们天空岛的冒险计划即将开启。无论何时何地，能够沉下心来，去打磨一张专辑，去雕琢一场舞台，去揣摩中华文化，都是一件难得的事。而他们正是怀抱着这样的初心，4月24日发布的夏日回归专辑先行曲《一杯火焰》中加入了自己创作的痕迹。

“前行，哪怕是黑云海啸，既然胸口装下了一整片海洋，眼睛里，自然倒映出一切超乎想象的波澜壮阔。”属于INTO1的追梦旅行，即将开启新的篇章，向长空启航。

伯远

Q：作为厨艺担当，上一个品尝到你厨艺的人是谁？
A：我好像很久没有做菜了，上一次做饭还是过年在家的时候，炒了个菜，这么说的话上一个品尝到我厨艺的应该是家人吧。

Q：即将发布组合的一周年单曲，为新歌做了哪些准备？
A：最近一直在准备新歌《一杯火焰》的录制，这首新歌有中英两个语言版本，我和刘宇、AK（刘彰）一起写了中文版的歌词。录中文版的时候，因为我自己有参与作词，就会有这一年印象比较深刻的瞬间在眼前回放的感觉，英文版则是代入我们海外成员的角度去唱，会想象他们录音时的心境。

Q：在创作单曲歌词的时候，想要给大家传递怎样的想法？
A：在创作周年纪念曲的过程当中，其实我有去想象听到这首歌的人们会是什么样的感觉，他们听到我们来描述这一年——这一年中我们成员之间关系的加深，从点头之交到渐渐熟悉，以及我们和喜欢我们的人之间的交流，我是怀抱着这样的心情写下的歌词。

Q：用三个词总结一下成团的这一年？
A：光阴似箭、愈发默契、吵吵闹闹吧。

Q：对你来说INTO1是什么？未来还有哪些想和成员们一起做的事情？
A：首先它是一份工作，这是事实，其次这是我一生中浓墨重彩的一笔。
人的一生中会有许许多多的经历，成为INTO1的一员，绝对是这其中非常特别的一段记忆。现在是我们的一周年生日，未来还有一年时间，好好珍惜吧，最想跟成员们一起做的事就是开演唱会。

Q：成团后感觉最有“团魂”的是哪次经历？
A：我觉得最有团魂的经历是在海边录制《打开INTO1的N种方式》的时候，我们在游泳池里面分成两队PK，虽然游戏中大家是对抗的，但是所有人都特别“嗨”、特别尽兴，真的能感受到我们是一个组合，团体感特别强。

Q：给自己新的一年立一个flag吧！
A：立太多flag我怕自己实现不了，其实只要能让关注我们的人对我们保持热情，我觉得就足够了，希望我们能做到让大家还想再次看到我们。

Q：不觉已经翻到毕业册的中途，如果可以对刚成团时的自己说一句话，你想说些什么？对一年后的自己呢？
A：对刚成团时的自己说，你的未来与你想象的一样，按你想象的去做；对未来的自己说，想象一下你的未来，就按想象的去做吧。
